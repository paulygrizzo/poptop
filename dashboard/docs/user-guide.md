# PopTop Dashboard User Guide

## Getting Started

### Accessing the Dashboard

1. Open the Google Sheet link (will be shared by Paul)
2. You'll see multiple tabs at the bottom:
   - **Dashboard** - Overview and key metrics
   - **Tasks** - All project tasks
   - **Phases** - Phase progress tracking
   - **Team** - Team member info
   - **Meetings** - Meeting notes
   - **Decisions** - Decision log
   - **Risks** - Risk tracking

---

## Daily Workflow

### 1. Check Your Tasks

1. Go to the **Tasks** tab
2. Filter by your name:
   - Click the filter icon in column D (Owner)
   - Select your name
3. Review tasks assigned to you
4. Update status as you work

### 2. Update Task Status

| Status | When to Use |
|--------|-------------|
| **Not Started** | Haven't begun work |
| **In Progress** | Actively working on it |
| **Complete** | Done! |
| **Blocked** | Can't proceed (add blocker details in column K) |
| **On Hold** | Paused for now |

### 3. Mark Tasks Complete

1. Change status to "Complete"
2. The "Completed" date will auto-fill
3. Add any final notes

## Adding New Tasks

1. Go to **Tasks** tab
2. Find the next empty row
3. Fill in:
   - **Phase** (select from dropdown)
   - **Task Name** (clear, actionable description)
   - **Owner** (who's responsible)
   - **Status** (usually "Not Started")
   - **Priority** (High/Medium/Low)
   - **Start Date** (when to begin)
   - **Due Date** (target completion)

**Tips:**
- Keep task names specific and actionable
- One owner per task
- Set realistic due dates

## Troubleshooting

### "I can't edit the sheet"
- Make sure you're signed into the correct Google account
- Contact Paul to verify your access

### "I'm not getting emails"
- Check your spam folder
- Verify your email is correct in the Team tab
- Make sure you're using @gmail or an email that accepts external mail

### "My changes disappeared"
- Google Sheets auto-saves, but check your connection

## Best Practices

1. **Update tasks daily** - Even a quick status update helps
2. **Be specific with blockers** - Say what you need
3. **Use realistic dates** - Better to underpromise and overdeliver
4. **Keep notes brief** - Key info only
5. **Ask questions** - Unclear? Reach out!
